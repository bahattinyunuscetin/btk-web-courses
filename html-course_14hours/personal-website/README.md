## 🧠 HTML Nedir?

**HTML (HyperText Markup Language)**, web sayfalarının iskeletini oluşturan temel işaretleme dilidir. Programlama dili değildir; web sayfalarının içeriğini yapısal olarak tanımlar. Tarayıcılar, HTML dosyalarını okur ve kullanıcıya anlamlı bir görsel arayüz olarak sunar.

### 📦 Temel Görevi:

* Web sayfasının içeriğini ve yapısını belirlemek
* Metin, görsel, bağlantı, form, tablo gibi ögeleri düzenlemek
* Tarayıcıya “ne nerede gösterilmeli?” talimatını vermek

---

## 🔍 HTML'in Yapısı ve Ana Etiketler

```html
<!DOCTYPE html>
<html>
  <head>
    <meta charset="UTF-8">
    <title>Sayfa Başlığı</title>
  </head>
  <body>
    <h1>Merhaba Dünya!</h1>
    <p>Bu bir paragraftır.</p>
  </body>
</html>
```

### 🧱 Yapı Taşları:

* `<!DOCTYPE html>`: HTML5 standardını belirtir.
* `<html>`: Tüm HTML belgesini saran kök etiket.
* `<head>`: Sayfa hakkında bilgi (meta), başlık (`<title>`), karakter seti.
* `<body>`: Kullanıcıya gösterilen içerik.

### 📚 Sık Kullanılan Etiketler:

| Etiket                        | Açıklama                          |
| ----------------------------- | --------------------------------- |
| `<h1>` - `<h6>`               | Başlık seviyeleri                 |
| `<p>`                         | Paragraf metni                    |
| `<a href="">`                 | Bağlantı (hyperlink)              |
| `<img src="">`                | Görsel ekleme                     |
| `<ul>` `<ol>` `<li>`          | Sırasız / sıralı liste öğeleri    |
| `<table>` `<tr>` `<td>`       | Tablolar ve satır hücreleri       |
| `<form>` `<input>` `<button>` | Formlar ve giriş alanları         |
| `<div>` `<span>`              | Genel kutulama ve grup etiketleri |

### 🧭 Semantik HTML:

HTML5 ile birlikte gelen bazı yeni, anlamlı (semantik) etiketler:

* `<header>`, `<footer>`, `<nav>`, `<section>`, `<article>` gibi etiketler, sayfanın bölümlerini anlamlı hale getirir.

## 🧪 Proje Türleri

Bu klasördeki projeler, temel HTML bilgilerini kavramaya yönelik aşağıdaki türlerde olabilir:

* **Kişisel tanıtım sayfası**: Başlık, paragraflar, görseller, iletişim bilgileri.
* **İletişim Formu**: Form elemanları (input, textarea, button).
* **Tablolu Bilgilendirme**: Ürün/kişi/konu bilgileri için tablo kullanımı.
* **Sıralı-Sırasız Listelerle Menü**: Navigasyon yapısı.
* **Basit Blog Şablonu**: Başlık, makale bölümü, yazar adı, tarih.

---

## 🎓 HTML Öğrenirken Dikkat Edilmesi Gerekenler:

* **Etiketlerin açılış ve kapanışına dikkat et!**
* **Yazım hataları (case sensitivity) önemli olabilir.**
* **Kapsayıcılık önemli: etiketlerin birbirine düzgünce "yuvalanmış" olması gerekir.**
* **Anlamlı içerik yapısı**, SEO ve erişilebilirlik (accessibility) açısından önemlidir.
